<h1>💫 React Made In Abyss Shop</h1>
<h6>♥️Демо-ролик проекта можно посмотреть в самом низу!♥️</h6>
<p>
	Это полноценный интернет-магазин стилизованный под моё любимое аниме Made In Abyss, с серверной и клиентской частью. React + Node js.
	Express для развертывания сервера и Sequelize ORM для работы с базой данных
</p>

<h2>🦊 Основные функции что выполнимы в проекте </h2>
<ul>
    <li>Полноценная регистрация и авторизация через JWT</li>
    <li>Фильтрация "товаров" как по "слоям" так и по категории контента</li>
    <li>Пагинация товаров</li>
    <li>Админ.Панель после авторизации где ты можешь добавлять любые категории/брэнды или товары</li>
    <li>Собственно,работа с базой данных, удаление/добавление etc.</li>
    <li>Корзина полноценная двух-этапная с учётом наличия/отсутствия в ней</li>
    <li>Красивая бесшовная аудио-анимация между страницами при подтверждении корзины(тематически на сайте не корзина а отправка в экспедицию где подтверждение это Отправиться в Экспедицию)</li>
    <li>Отдельные страницы генерируемые под каждый "товар"</li>
</ul>

<h2>🪝 Как локально запустить этот проект у себя</h2>
<ul>
    <li>1. Клонировать репозиторий</li>
    <li>2. Установить зависимости в серверную папку и клиентскую через: npm install</li>
    <li>3. Настроить .env файл для сервера и для клиента, пример: <br> <img width="252" height="213" alt="image" src="https://github.com/user-attachments/assets/114e0b70-ee83-446a-9235-d8488ab1e79b" /></li>
    <li>4.Откройте pgAdmin 4 и создайте новую пустую базу данных:Servers → PostgreSQL → Databases → Create → Database...</li>
    <li>5.Импортируйте базу данных из файла server/database/online_store.sql <br>Вариант 1 — через pgAdmin: <ul><li>
        
<li>Откройте pgAdmin 4</li>
<li>Выберите созданную базу данных</li>
<li>Нажмите Tools → Query Tool</li>
<li>Откройте файл:server/database/online_store.sql</li>
<li>Выполните SQL-скрипт кнопкой Execute / ▶</li>
<li>После этого структура базы данных и тестовые данные будут импортированы.</li>
    </li></ul></li>
    <li>5. После всего этого остаётся запустить в терминале из папки server: npm run dev</li>
    <li>6. В client: npm start</li>
</ul>
<h2>⌛ Возможные проблемы</h2>

<h3>Ошибка подключения к БД</h3>
<p>Проверьте:</p>
<ul>
  <li>запущен ли PostgreSQL</li>
  <li>совпадают ли значения в <code>.env</code></li>
  <li>существует ли база данных</li>
  <li>успешно ли выполнился SQL-файл</li>
</ul>

<h3>Ошибка <code>role does not exist</code></h3>
<p>
  У вас может быть другой пользователь PostgreSQL.
  Измените <code>DB_USER</code> в <code>.env</code> и команду импорта.
</p>

<h3>Не запускается backend</h3>
<p>Убедитесь, что зависимости установлены:</p>
<pre><code>npm install</code></pre>

<h3>Порт уже занят</h3>
<p>
  Измените <code>PORT</code> в <code>.env</code> или освободите занятый порт.
</p>
